# Kazumi TV

使用 Flutter 开发的基于自定义规则的番剧采集与在线观看程序。使用最多五行基于 `Xpath` 语法的选择器构建自己的规则。支持规则导入与规则分享。支持基于 `Anime4K` 的实时超分辨率。

> 本仓库为 [Kazumi](https://github.com/Predidit/Kazumi) 的 Android TV 适配版本，仅发布 Android TV 平台的安装包。

## 支持平台

- Android TV (Android 10 及以上)
